# Coding Conventions

## Language Standards
- **Python 3.13**: Targeted runtime.
- **PEP 8**: General adherence to standard Python styling.

## Patterns & Practices
- **Environment Management**: Configuration via `.env` file and `python-dotenv`.
- **Asynchronous Execution**: Heavy use of `asyncio` and `aiohttp` for non-blocking operations.
- **Modularization**: Code is organized into logical packages and sub-packages.

## Naming Conventions
- **Files**: Lowercase with underscores (`main.py`, `setup.py`).
- **Directories**: Lowercase (e.g., `assistant`, `core`).
