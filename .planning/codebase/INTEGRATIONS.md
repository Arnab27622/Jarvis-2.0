# External Integrations

## AI & LLM Services
- **Groq**: High-performance LLM inference.
- **OpenRouter**: Unified API for various LLM models.
- **Hugging Face**: Inference API for open-source models.
- **Stability AI**: Image generation services.
- **Decohere AI**: Specialized AI features.

## Data & Search APIs
- **YouTube Data API v3**: Video search and playback information.
- **OpenWeatherMap**: Weather updates and forecasts.
- **SerpAPI**: Advanced web searching via Google/other engines.
- **News API**: Daily news headlines.
- **Wikipedia**: Knowledge base queries.

## Hardware & OS Integrations
- **Windows API (via WMI, pywin32)**: System control (brightness, volume, windows).
- **Audio I/O**: Microphone input and speaker output for voice interaction.
